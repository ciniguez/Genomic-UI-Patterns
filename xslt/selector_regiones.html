      Especificar regiones de interés dentro del genoma.

      Fig. 2 Opciones del filtro “Genomic Locus” de la aplicación GENESIS 2.0.[1].

      Problema

      Especificar regiones de interés dentro del genoma.

      Contexto

      • El usuario quiere enfocarse en una región o regiones concretas dentro del genoma.
      • El usuario conoce las posiciones o coordenadas exactas (locus del genoma) de las regiones que quiere visualizar o explorar.

      Solución

      Permitir al usuario indicar el número de cromosoma y, dentro de él, la región de interés especificando el locus inicial (posición inicial) y el locus final (posición final). Una alternativa más elaborada de este mecanismo de selección puede permite especificar varios cromosomas y las coordenadas de inicio y fin se aplican para todos los cromosomas seleccionados (como lo muestra la Fig. 1). Para usuarios con poca experiencia computacional se puede proveer controles de desplazamiento (p.ej. slider) que permiten manualmente especificar el rango de interés, sin embargo, esta opción no es recomendada para especificar posiciones precisas.

      Esquema

      La flecha en la parte inferior señala un cuadro que remarca el control de interfaz de usuario utilizado para configurar la región de la secuencia del genoma que se quiere visualizar.
