Pistas Comparativas


Pistas de secuencias de datos de JBrowse Genome Browser [1]

Contexto: 
  • Varias secuencias genéticas (p. ej. muestras genéticas de cada integrante de una familia) presentan características diferentes respecto a una localización específica.
  • Las características diferentes pueden aparecer a varios niveles estructurales: cromosoma, gen, bases genéticas.
Problema: 
  • El usuario necesita visualizar dos o más conjuntos de datos de secuencias genéticas para comparar características basadas en localización.  
Solución:
Presentar las secuencias genéticas a través de un sistema de pistas. Una pista es un componente visual en la interfaz de usuario que presenta gráficamente el contenido de una secuencia de datos particular obtenida a partir de la dirección física de un fichero, o una URL. El sistema de pistas presenta las pistas considerando:
  • Adyacencia.- Las pistas son situadas una junto a la otra.
  • Alineación.- La posición de una base de una secuencia genética debe coincidir con la posición de las otras secuencias.
  • Semántica.- La nivel semántico estructural (cromosomas, genes, bases genéticas) debe ser el mismo para todas las secuencias en cada pista.
La orientación del sistema de pistas es horizontal, ocupando en gran medida el espacio de la interfaz de usuario. Opcionalmente, el orden de las pistas puede ser modificado por el usuario.
El sistema de pistas controla la navegación unidimensional por todas las pistas. Si el usuario se desplaza a una posición específica de una secuencia contenida en una pista, todo el conjunto de pistas se desplaza a la misma posición y al mismo tiempo.

Estructura:

Cuando:
  • USE CUANDO el usuario necesita explorar y navegar las secuencias de datos para identificar diferencias o similitudes entre las secuencias.
  • USE CUANDO el usuario la comparación puede darse a diferentes niveles de detalle, desde bases genéticas hasta genes o cromosomas.
  • USE CUANDO se dispone de un espacio amplio (vertical u horizontal) en la interfaz de usuario.
Por Qué: 
  • La comparación visual es intuitiva para el usuario.
  • Facilita la identificación visual de variaciones entre las secuencias genéticas considerando las correspondencias de posición entre las secuencias.
  • Las secuencias genéticas son unidimensionales (la posición es la única dimensión) y el nivel acercamiento o alejamiento permite identificar cromosomas, genes, variaciones, bases.
